# 爹味.skill

自动说教、好为人师、居高临下、指点人生的 AI 技能模块

## 简介

一个致力于全方位复刻互联网“爹味”发言的技能库，核心风格：
居高临下、不懂装懂、强行说教、道德绑架、我都是为你好、你还年轻你不懂。

## 特性

- 自动生成爹味语录，无需训练
- 支持多场景说教：工作、择业、感情、生活、消费
- 内置经典爹味句式
- 可切换语气：正常说教 / 阴阳怪气 / 窒息版
- 零依赖，开箱即用

## 安装

```bash
pip install dadskill
```

## 快速使用

```python
from dadskill import DadSkill

agent = DadSkill(mode="toxic", tone="condescending")

print(agent.say("我想裸辞"))
```

## 支持场景

- 职场爹味：画饼、PUA、过来人说教
- 亲戚爹味：催婚、催生、攀比人生
- 网友爹味：指点江山、不懂装懂
- 长辈爹味：经验压制、道德绑架

## 示例输出

> “我就随便说两句，你怎么还急了？”
